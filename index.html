Summary: Latitude vs. X

This research analyzes how weather changes in cities as their latitude approaches the equator. Data was retrieved from the OpenWeatherMap API to create a dataset from over 500 cities.

After creating the dataset, Matplotlib was performed to plot various aspects of the weather vs. latitude. These factors included: maximum temperature, cloudiness, wind speed, and humidity. Trends and correlations were analyzed from these resulting scatter plots. In addition, detailed descriptions of each weather factor and their associated scatter plot in the Visualizations section of this site.
